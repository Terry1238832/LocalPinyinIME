# Installer

`LocalPinyinIME.iss` is a legacy Inno Setup script for experiments. The
recommended release path is the reproducible ZIP package generated by
`scripts/Build-Release.ps1`.

The Inno script does not carry an independent version string. `Build-Release.ps1`
reads `cmake/LocalPinyinReleaseVersion.cmake` and writes
`dist/LocalPinyinIME-InnoVersion.iss`, which the Inno script includes.

The Inno script installs to a versioned x64 directory and calls
`LocalPinyinImeSetup.exe` for registration and unregistration. It does not use
silent `regsvr32`.

Run any installer with administrator privileges. Do not use this script to
claim a signed or final release.
